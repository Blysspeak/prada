# @blysspeak/prada-ui

React-based admin panel interface for PRADA.

## About

This package contains the pre-built UI for PRADA admin panel. It's automatically included when you use `@blysspeak/prada-server` or the `prada` CLI.

## For Development

If you want to customize the UI:

```bash
git clone https://github.com/blysspeak/prada.git
cd prada/packages/ui
npm install
npm run dev
```

## Tech Stack

- React 18
- TypeScript
- TailwindCSS
- React Query
- React Router
- React Hook Form
